# 2.1 Fundamental xizmatlar (Broker bilan qo'shni xizmatlar)

## 2.1.0 Kirish

Fundamental xizmatlar - sizning biznes g'oyangiz va brokerlar o'rtasidagi munosabatni ta'minlovchi ilk qobiqdir.

Umuman olganda, broker bilan qo'shni xizmatlar (fundamental xizmatlar) biznes g'oyaning gibrid ko'rinishi bo'lib, yuqori darajadagi biznes g'oya bajarilganida abstarakt qatlam vazifasini o'tab beradi. Yuqori darajadagi biznes g'oya haqida keyingi bo'limda batafsil gaplashib o'tamiz. 

Fundamental xizmatlarning asosiy vazifasi tizimga kiruvchi va undan chiquvchi ma'lumotlarning to'g'riligi va tizim talablariga muvofiqligini tekshirishdan iborat.

Shuningdek, fundamental xizmatlar, brokerlar taklif qilayotgan asosiy operatsiyalar ustiga qo'shimcha tasdiqlash qatlamini taqdim etadi.

Misol uchun, agar saqlash brokeri `InsertStudentAsync(Student student)` deb nomlangan metodni takfil qilsa, fundamental xizmat quyidagi kodni taqdim qiladi:

```csharp
public async ValueTask<Student> AddStudentAsync(Student student)
{
	ValidateStudent(student);

	return await this.storageBroker.InsertStudentAsync(student);
}
```

Bu shuni ko'rsatadiki, fundamental xizmat broker taklif qilayotgan mavjud operatsiya ustiga qo'shimcha tasdiqlash operatsiyasi qatlamini qo'shishdan boshqa hech qanday ish bajarmaydi.
